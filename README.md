# 個人用辞書のUI

UI for [dictionary][https://github.com/hrdrq/dictionary]

## パソコンとスマホのブラウザとGoogle Chrome拡張機能が使える

node -v 11.14.0

npm -v 6.7.0


## Build Setup

``` bash
# dependenciesをインストール
$ npm install

# 開発環境
$ npm run dev

# 本番
$ npm run build

# Google Chrome拡張機能
$ npm run gc
```
