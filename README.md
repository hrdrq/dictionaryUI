# 個人用辞書のUI

## パソコンとスマホのブラウザとGoogle Chrome拡張機能が使える

## Build Setup

``` bash
# install dependencies
$ npm install

# 開発環境
$ npm run dev

# 本番
$ npm run build

# Google Chrome拡張機能
$ npm run gc
```
